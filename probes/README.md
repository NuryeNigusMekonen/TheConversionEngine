# Probe Artifacts

This directory contains:

- `adversarial_probe_library.json`
- `probe_library.md`
- `failure_taxonomy.md`
- `target_failure_mode.md`

`adversarial_probe_library.json` is the canonical structured probe artifact.
`probe_library.md` is the human-readable index and category coverage summary.

The probes focus on Tenacious-specific risk: ICP precedence, signal honesty, bench over-commitment, tone drift, multi-thread leakage, channel hierarchy, scheduling, public-signal lossiness, and competitor-gap over-claiming.
